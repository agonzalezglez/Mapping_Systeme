# NGS Project: SAM File Processing

## Introduction
This project focuses on the analysis of SAM (Sequence Alignment/Map) files, a standard format used to store genomic sequence alignment data. It aims to extract and interpret key information from these files, including the characteristics and occurrence of alignment flags, in order to better understand the structure and content of the data.

## Prerequisites
Before you can run this project, you need to have certain tools and programs installed on your machine.

- Linux or similar : This project is designed to run on Linux.
- Python 3 : Used to execute scripts.
- Pip : Used to manage Python dependencies.

## Installation

### Import of the project
To install all the files in our project, you need to enter the following line of code in your terminal:
```bash
git clone https://github.com/agonzalezglez/Mapping_Systeme.git
```
### Installation of packages:
In order to download the matplotlib package used in our program, you must enter the following line of code in your terminal:
```bash
pip install matplotlib
```
## How to use the script?
To launch the script, enter the following line of code in your terminal:
```bash
./project_sam.py
```
A message asking you to enter the path of your sam file will show up on teh terminal ('Enter the path to your SAM file:').
